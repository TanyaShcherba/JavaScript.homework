Создать функцию changeArray, которая будет менять местами половины массивов, т.е. если количество элементов четное, то вторая половина становится сначала массива, а первая на место второй; если количество элементов нечетное, тогда элемент массива, который является серединой остается на месте, а половины меняются местами.

Например:
changeArray([ 1, 2, 3, 4, 5 ]); //[ 4, 5, 3, 1, 2 ]
changeArray([ 1, 2 ]); //[ 2, 1 ] 
changeArray([ 1, 2, 3, 4, 5, 6, 7, 8]);  //[ 5, 6, 7, 8, 1, 2, 3, 4 ]
